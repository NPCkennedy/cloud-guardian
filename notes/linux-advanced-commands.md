grep - search inside files
Example: grep "Owner" resource_scanner.py
This searches for the word Owner inside your scanner file

find - find files by name
Example: find . -name "*.py"
This finds all Python files in the current directory

curl - download from the internet or test APIs
Example: curl https://www.google.com
This downloads the Google homepage into your terminal

tar - compress and extract files
Example: tar -czf backup.tar.gz notes/
This creates a compressed backup of your notes folder

chmod - change file permissions
Example: chmod +x myscript.py
This makes a file executable

systemctl - manage system services
Example: systemctl status ssh
This checks whether the SSH service is running
